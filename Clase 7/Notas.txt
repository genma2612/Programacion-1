Comandos Git Bash:
	git init  // inicia nuevo repositorio.
	git clone // clona repositorio.
	git status // scan unstaged files
	git add . //  agrega todos los archivos
	git add "nombre de archivo" // agrega archivo
	git commit -m "mensaje" // commit sin mensaje por consola unix.
	git push origin // Push
